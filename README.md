
# Bikeshare Project

## Overview
This project explores US bikeshare data for three major cities: Chicago, New York City, and Washington.  
The program allows users to interactively filter the data by city, month, and day of the week, then displays descriptive statistics based on the selected filters.

## Files Used
- **bikeshare.py**: Python script that loads and analyzes bikeshare data
- **chicago.csv**: Bikeshare data for Chicago (ignored by Git)
- **new_york_city.csv**: Bikeshare data for New York City (ignored by Git)
- **washington.csv**: Bikeshare data for Washington (ignored by Git)
